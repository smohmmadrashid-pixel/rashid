## ✨ Features & UI Highlights

* **Industry-Tailored Design:** Built with a high-visibility, crisp color palette (Safety Orange, Deep Slate Blue, and Clean White) that aligns perfectly with industrial health and safety standards.
* **Fully Responsive Adaptability:** Seamless navigation across all screen sizes, featuring a custom interactive mobile slide-out menu.
* **Dynamic Credentials Interface:** An interactive tab-filtered system separating core institutional certifications from specialized field trainings.
* **Built-in Graphics:** Custom SVG safety illustrations and pure CSS animations integrated directly into the code—ensuring rapid load times and zero broken image paths.
* **Skill Analytics:** Visual data progress bars highlighting core specializations like Permit to Work (PTW) systems and hazard inspections.

---

## 🛠️ Tech Stack Used

* **HTML5:** Structured semantic markup for clean accessibility and search engine optimization.
* **CSS3:** Custom layout system with variables, interactive flexbox/grid alignments, responsive media queries, and keyframe floating animations.
* **JavaScript (ES6):** Powering mobile menu toggles, interactive tab filtering, and client-side form submission handling.
* **FontAwesome (v6.4.0):** Rich vector icons for professional visual indicators throughout the interface.
